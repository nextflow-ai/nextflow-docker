## 🚀 Bắt đầu nhanh

### 1️⃣ Thiết lập môi trường
```bash
# Cài đặt Python dependencies
pip install -r requirements-ai.txt

# Khởi động AI services
docker-compose up ai-engine
```

### 2️⃣ Cấu hình AI features
```python
# Kích hoạt AI features trong CRM
from nextflow.ai import enable_ai_features

enable_ai_features([
    'lead_scoring',
    'sentiment_analysis', 
    'chatbot',
    'predictive_analytics'
])
```

### 3️⃣ Training models
```python
# Huấn luyện model với dữ liệu của bạn
from nextflow.ai.training import train_model

train_model(
    model_type='lead_scoring',
    data_source='crm_database',
    validation_split=0.2
)
```
